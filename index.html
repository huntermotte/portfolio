Portfolio

yNot

yNot allows you to get random suggestions for restaurants and bars in your area, wherever you are. When your spouse asks where you want to go for dinner, when your friends ask what bar you want to go to, when your coworkers want to know which lunch spot to hit, yNot puts venues into your mind which you may not have thought of otherwise.

Built with React Native + Redux. Available on iOS and Android.

Open Source Trading

Open Source Trading a project that allows users to create securities trading ideas and interact with other traders to open source knowledge. You can create and delete your own ideas and explore what others have to say. I love researching and trading stocks, so I created Open Source Trading to be able to gather my random thoughts in a journal of sorts. Of course, the more investors that join, the more open source trading information is available!

Built with HTML, CSS (utilizing Bootstrap), JavaScript/jQuery, MongoDB, Express.js, Node.js, and Passport for authentication. API is my own.

Fridge Cleaner

Fridge Cleaner is an app that allows users to search any ingredients and find recipes that use them. I found myself constantly stuck with random ingredients in my fridge or pantry without a clear plan of what to do with them before they expired. Fridge Cleaner helps alleviate that concern, utilizing Edamam's API to retrieve the comma-delineated ingredients and find matching recipes!

Built with HTML, CSS, JavaScript/jQuery, and third party API, Edamam (https://developer.edamam.com).

Hangout Roulette

Hangout Roulette is a way to simplify the age-old question "Where should we go to eat?" Initially built around my current location in downtown Raleigh, the app utilizes FourSquare's API to pull a random venue's data and present that as an eating spot. Users can save their favorite venues and add notes, so the next time a venue shows up they can see what they thought about it last time. With so much good food in Raleigh, it seemed like a no-brainer, especially for someone who loves to try new spots.

Built with React.js, Redux, CSS (no Bootstrap, just media queries), MongoDB, Express.js, Node.js, and Passport for authentication. Google Maps API (https://developers.google.com/maps) and Foursquare API (https://developer.foursquare.com) used to generate venue data and map data. Backend API is my own.
